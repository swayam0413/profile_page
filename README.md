# Swayam Patel Portfolio

Modern personal portfolio built with React, Vite, TypeScript, Tailwind CSS, Framer Motion, and Lucide React.

## Run Locally On Windows

Double-click:

```text
start-portfolio.bat
```

Then open:

```text
http://127.0.0.1:5173/
```

Or run from PowerShell:

```powershell
npm.cmd run dev -- --host 127.0.0.1 --port 5173
```

If `npm` fails in PowerShell with a script execution policy error, use `npm.cmd` as shown above.

## Build

```powershell
npm.cmd run build
```

The production output is created in `dist`.

## Update Links

Edit `src/App.tsx` to replace the placeholder GitHub and LinkedIn URLs with Swayam's real profile links.

To make the resume button work, add the resume PDF here:

```text
public/Swayam-Patel-Resume.pdf
```
